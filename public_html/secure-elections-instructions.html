Secure Elections

Thanks for stopping by! These instructions are for those who are looking for a way to make a ranked choice ballot fast, easy, inexpensive, and secure while using RankedChoices.com!

I recommend registering an account first, then the organizer can easily remove any fraudulent votes.

Open Election:

Everyone's vote is public. The only concern here is if a different person voted under someone else's name, it would be hard to identify who the real vote is vs the fake.

  1. Make a list of randomly generated numbers.
  2. Assign a number to each person not showing others.
  3. Choose "Voter Names Required" from the advanced options.
  4. Instruct each voter to enter that number next to their name.
  5. Use advanced options to release results after voting is cut-off.

By taking this approach, the organizer can verify that the name of the voter and their assigned number match if there are any disputes.

Blind Election:

Everyone's vote is hidden from other voters, but not the organizer. The concern here is not allowing someone to anonymously vote twice.

Option 1:
  1. Choose "Voter Names Required" from the advanced options.
  2. Choose "Hide voter names when displaying results (except from owner)" from the advanced options.
Option 2:
  1. Make a list of randomly generated numbers.
  2. Assign a number to each person not showing others.
  3. Choose "Voter Names Required" from the advanced options.
  4. Use advanced options to release results after voting is cut-off.
  5. Instruct each voter to enter that number INSTEAD OF their name.

By taking this approach, the organizer can verify that every vote is legitmate because if an entered number does not match the list of generated numebrs, the vote can be thrown out.

Double-Blind Election:

Everyone's vote is hidden from other voters, including the organizer. The concern here is not allowing someone to anonymously vote twice nor allowing any way to trace a vote back to the voter.

  1. Make a list of randomly generated numbers.
  2. Copy the list of randomly generated numbers and separate them (e.g. cut them out and place them into a bowl).
  3. Allow each voter to secretly receive their number without others knowing (e.g. discretely drawing from a bowl).
  4. The voter keeps this number a secret during the entire election and beyond.
  5. Choose "Voter Names Required" from the advanced options.
  6. Use advanced options to release results after voting is cut-off.
  7. Instruct each voter to enter that number INSTEAD OF their name.

By taking this approach, the organizer can verify that every vote is legitmate because if an entered number does not match the list of generated numbers, the vote can be thrown out.

Also, the organizer has no way of tracing a number back to a voter without that voter's concent.
